---

## 📌 프로젝트 소개

향기 저장소 (PerfumeLog)는 사용자들이 다양한 향수에 대한 리뷰를 공유하는 커뮤니티 기반 플랫폼입니다.

### 🎯 제작 목표

- 사용자 리뷰 기반 향수 정보 공유 플랫폼 구축
- 향기 노트, 지속력, 잔향 등 향수 특화 정보 기반 리뷰 시스템 제공

---

## ⚙️ 기능

| 구분       | 기능 설명 |
|------------|-----------|
| 사용자     | 회원가입 / 로그인 |
| 검색       | 향수명, 브랜드명, 향 노트 기반 검색 |
| 향수 목록  | 브랜드, 향 계열 필터링 |
| 상세 정보  | 향기 노트, 지속력, 정가, 리뷰 등 상세 보기 |
| 리뷰       | 리뷰 작성, 평점 등록 |
